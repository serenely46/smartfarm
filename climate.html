매핑 규칙 & 저장 동작
  • 주간/야간 기준: 외부일사가 0에서 양수로 바뀌면 주간 시작, 양수에서 0으로 떨어지면 야간 시작
  • 외부누적 광량(J/cm²): 외부일사(W/m²) × 60초 ÷ 10,000 의 일별 합계
  • 주차 단위: 일요일 시작 기준 7일씩 묶어 주평균(노란색 칸) 자동 계산
  • 🛡 안전 저장 (기본): 매그마 데이터를 저장할 때 기존 날짜는 자동으로 보호됩니다. 같은 날짜의 DB 데이터는 덮어쓰지 않고 새 날짜만 추가합니다.
  • ⚠ 강제 덮어쓰기: "기존 날짜도 덮어쓰기" 체크 시에만 기존 데이터를 새 값으로 갈아치웁니다 (확인 팝업 한 번 더 떠야 진행).
  • 시각 표시: 🟢 초록 테두리 = 새로 추가될 셀 / 🔴 빨간 테두리 = 덮어쓰기 모드일 때 사라질 기존 셀 / 흐린 셀 = DB에 있는 기존 데이터

수분관리일지 도움말
  • 입력 항목 (노란 셀): 시간 4개, 1회공급량, 총공급횟수, 배액량, 공급EC, 슬라브 EC, 슬라브 평균/최고/최저 함량
  • 자동 계산 (회색 셀): 실제 총 공급량 = 1회공급량 × 공급횟수, 배액량%, 슬라브 수분 편차 등
  • 외부누적 광량: 환경관리 탭에서 저장된 데이터를 자동으로 끌어옵니다
  • 🆕 100J 도달 시간: 매그마 파일 업로드 시 자동 계산 — 분 단위 누적 광량이 처음 100J/cm²에 도달하는 시간
  • 🆕 잔광 200J 남는 시간: 매그마 파일 업로드 시 자동 계산 — (그날 총 누적 광량 − 200J)에 도달하는 시간 (즉 일사 종료 200J 전 시점)
  • 매그마 → 수분일지 자동 입력: 환경관리 탭에서 매그마 파일을 업로드하면 두 시간 값이 자동 계산되며, 환경관리 저장 시 수분일지에도 함께 저장됩니다. 비어있는 셀에만 채워지므로 직접 입력한 값은 보존됩니다.
  • 저장: 입력 후 💾 버튼 또는 입력칸 밖 클릭 시 자동으로 변경 표시되며, 저장 버튼으로 일괄 Supabase 반영
